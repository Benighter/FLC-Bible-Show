### Prerequisites
- Java 17 or higher
- Gradle (included via wrapper)

### Build Instructions
```bash
# Clone the repository
git clone https://github.com/Benighter/FLC-Bible-Show.git
cd FLC-Bible-Show

# Build the application
./gradlew build

# Run the application
./gradlew run
```

### Creating Distribution
